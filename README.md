# Employee-Maintenance-System
Full Stack Crud Application for Employee Management using SpringBoot and React
Backend - Core Java, Hibernate, SpringBoot. In future Microservices would be applied.
FrontEnd - React-18 : Used React Routing to enroute all REST API's from App.jsx and modified Service layer to be able to capture request response from UI and backend.
Usage of Hooks to populate backend response properly in UI.
Added Spring Security for Secure login, logout with persistency maintained by Sql.
